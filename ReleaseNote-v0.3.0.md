# SpaceKatMotionMapper v0.3.0 更新日志

本次 v0.3.0 版本带来了大量的底层重构、交互界面的大更新以及万众期待的基础 Linux 平台支持。

## 🎉 新增特性 (New Features)
- **动作选项卡分组配置**：大幅简化设置流程，现在可以更直观、有条理地根据类别与分组配置各项动作。
- **全新单动作模式**：
  - **推动直接触发**：简化操作交互，即推即触发。
  - **单键自动重复**：符合日常长按单键的使用直觉。
  - **组合键默认触发一次**：防止重复触发导致的不必要误触与失效。
- **智能配置冲突检测**：新增健壮的配置冲突与重叠检测，在遇到组合键或配置项冲突时会给出明确的 UI 错误提示，帮助快速排查错误。
- **Linux 平台初步支持**：进行底层解耦使其支持跨平台。初步支持了 Linux 环境，当前首发适配 **Niri** 窗口管理器（含前台应用识别、最小化辅助悬浮窗等，需配合 `ydotool` 进行按键模拟）。
- **设备支持扩展**：新增适配 **MiniE 有线模式**，为未来兼容 Mini2 等设备打下了良好的基础。

##  进阶优化与调整 (Enhancements)
- **UI 与框架全面焕新**：
  - 核心前端框架已平滑升级至 **Avalonia 12**。
  - 大幅重构、美化主页面及配置选择对话框界面，加入快捷悬浮控件，提高可读性并优化跨系统分辨率（DPI）适配缩放。
- **语义校验拦截**：新增严密的按键动作组合校验管线（如按下与释放平衡检测），避免编写出无意义或引发死锁的组合宏。
- **可靠的热键录入体验**：重构热键输入框体底层逻辑，修正了焦点丢失和录入识别偏差，添加组合按键更加跟手。
- **日志与底层架构革新**：全面迁移日志分析系统至 **Serilog** 与 **OpenTelemetry** ，优化程序依赖注入流程。
- **运行时环境升级**：适配 .NET 全新运行时版本（涉及 .NET 9/.NET 10 生态），确保极速启动体验。

## 🐛 修复 (Bug Fixes)
- 修复了版本迁移时如果遇到未知旧版本配置时未能正确抛出和捕获异常的问题。
- 修复了“第一次运行时加载高级配置”可能导致的一系列状态未同步 BUG。
- 修复优化了热键时间相关设置（如死区范围下限）可能造成的隐式配置冲突或 UI 卡顿。
